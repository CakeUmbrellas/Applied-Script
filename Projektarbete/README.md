# Check Password

Ett script i Python som kollar upp angivna lösenord i databaser hos "Have I Been Pwnd" och "rockyou.txt".

## Beskrivning

Scriptet hämtar data från "passwords.txt", delar på username och password, kopplar upp sig till "Have I Been Pwnd" först och sedan till "rockyou.txt" för att se om de angivna lösenorden finns läckta. Sedan skrivs resultatet ut.

## Getting Started

### Dependencies

* Scriptet måste köras från Linux.
* Python måste vara installerat.

### Installing

* Ladda ner repot.

### Executing program

* Kör filen från korrekt mapp.

## Tips

* Ändra textfilens lösenord för att se olika resultat.
